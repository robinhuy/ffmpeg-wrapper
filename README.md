# FFMPEG Wrapper
Desktop Application base on FFMPEG

## Install
1. Install dependencies

```
    $ npm install
```

2. Run app

- On Windows:

```
    $ export OS=windows && npm start
```

- On Linux:

```
    $ export OS=linux && npm start
```

- On Mac:

```
    $ npm start
```

## Build app

1. For Windows:

```
    $ npm windows
```

2. For Linux:

```
    $ npm linux
```

3. For Mac:

```
    $ npm mac
```
