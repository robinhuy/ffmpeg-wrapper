# ffmpeg-wrapper
Desktop Application base on FFMPEG

## Install
1. Install dependencies

```
    $ npm install
```

2. Run app in development mode

- On Windows:

```
    $ export OS=windows && npm start
```

- On Linux:

```
    $ export OS=linux && npm start
```

- On Mac:

```
    $ npm start
```

3. Build app

- For Windows:

```
    $ npm windows
```

- For Linux:

```
    $ npm linux
```

- For Mac:

```
    $ npm mac
```
